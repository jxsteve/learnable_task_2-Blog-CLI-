## Blog CLI Application

This project is a simple command-line tool that lets you manage blog posts directly from your terminal.

### What it does

You can create, view, edit, delete, and search blog posts without needing a web interface. It’s designed to be lightweight and easy to use.

### How it works

The application runs in the terminal and stores posts locally (for example, in a file or small database). Each post includes a title, content, and optional tags, and can be saved as a draft or published.

### Why build it

This project helps you practice:

* Working with command-line interfaces (CLI)
* Handling user input
* Managing data (create, read, update, delete)
* Structuring a small application

### Typical features

* Add new blog posts
* List all posts
* View a specific post
* Edit or delete posts
* Search by keyword or tags

### Goal

The goal is to build a simple, functional tool that demonstrates core programming concepts while being useful and easy to extend later.
